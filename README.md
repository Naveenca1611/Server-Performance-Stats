# Server Performance Statistics Script

A simple Linux shell script to monitor and analyze basic server performance statistics.

## Features

- CPU Usage
- Memory Usage
- Disk Usage
- Top 5 CPU-consuming processes
- Top 5 Memory-consuming processes
- Hostname & OS details
- System Uptime

---

- Linux
- Shell Scripting (Bash)
- Ubuntu EC2 Instance
